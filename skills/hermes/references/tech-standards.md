# ATT Standard Technology Standards

This is ATT Infosoft's standard list of technology standards referenced in technical proposals.
Update for specific tenders where the RFP mandates specific versions or additional standards.

## Networking
- Transport Control Protocol / Internet Protocol (TCP/IP)

## Web
- HTTP 1.1
- Hypertext Transport Protocol Secure (HTTPS)
- File Transfer Protocol (FTP)
- HTML 5.0 (or 4.0 where legacy systems apply)
- XML 1.0
- SOAP 1.1 or 1.2
- WSDL 1.1 or 1.2
- REST API (JSON over HTTPS) — preferred for modern integrations
- UDDI 2.0

## Security
- Secure Socket Layer (SSL) 3.0 / TLS 1.2+
- X.509 v3 certificate
- WS-Security 1.0
- OWASP Top 10 compliance

## Messaging
- Simple Mail Transport Protocol (SMTP)
- Secure SMTP
- Multipurpose Internet Mail Extensions (MIME)

## Directory Services
- LDAP version 3
- Active Directory Service Interface (ADSI) 2.5
- Java Naming and Directory Interface (JNDI) in J2EE 1.4 or higher

## Database Access
- Open Database Connectivity (ODBC) 3.51 or higher
- Java Database Connectivity (JDBC) 2.0 or higher

## Application Integration
- Java Native Interface (JNI) in J2EE 1.4 or higher
- Java Message Service (JMS) in J2EE 1.4 or higher
- J2EE Connector Architecture (JCA) in J2EE 1.4 or higher
- RESTful APIs (JSON/HTTPS) for modern system integration

## Client-Side Technologies
- HTML 5.0 / XHTML 1.0 or above
- XML 1.0 or above
- DOM Level 1 or above
- CSS Level 3 or above
- JavaScript (ES6+)
