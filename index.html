Project Atlas
An AI agent system built to handle IT tender proposals end-to-end. Starting with technical specification writing, Atlas will be trained with company knowledge and purpose-built skills to produce competitive, high-quality tender submissions.

Objectives

Immediate goal

Train Atlas to read an RFP and produce a structured technical specification section. Evaluate against past ATT submissions and iterate via a feedback loop.

Long-term vision

Atlas orchestrates multiple sub-agents, each owning a tender section skill. Output is a complete, review-ready proposal assembled from specialist agents.

System Design
Architecture
Atlas runs a multi-agent architecture. The orchestrator reads the RFP, assigns tasks to sub-agents by skill, and assembles the final output. Model routing is cost-optimised across three tiers.

Tender Skills
Skills Roadmap
One skill per tender section. Each skill is an independent module assigned to a sub-agent. Atlas activates skills based on the RFP. Skills improve through the feedback loop over time.

Reads RFP and generates detailed technical approach, system architecture description, and compliance mapping for the technical specification section.
